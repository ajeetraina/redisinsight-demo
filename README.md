# redisinsight-demo


# Getting Started

## Install Redis Stack on your Mac

Two different ways

- Either download manually
- Using brew

```
brew tap redis-stack/redis-stack
brew install --cask redis-stack
```

## Start Redis Stack server

```
 redis-stack-server
 ```
 
 ## Test the connectivity
 
 ```
 redis-cli
 ```
